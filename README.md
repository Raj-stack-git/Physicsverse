# Physicsverse - Electrostatics Visualizer

## 🚀 Interactive Physics Simulation Engine

A premium dark-themed, interactive electrostatics visualizer built with React, Three.js, and Tailwind CSS.

### ✨ Features

- **Module I: Coulomb's Law & Superposition** - Place and manipulate multiple point charges, visualize force vectors dynamically
- **Module II: Electric Field Lines** - Visualize field lines for single charges, dipoles, and charge configurations
- **Module III: Gauss's Law in 3D** - Interactive 3D Gaussian surfaces with flux visualization
- **Module IV: Capacitance & Dielectrics** - 3D parallel plate capacitor simulator
- **Real-time Physics Engine** - Dynamic force calculations using Coulomb's law
- **Premium UI** - Dark theme, glassmorphic components, neon accent colors (Cyan/Magenta)
- **NCERT Integration** - Embedded revision cards and derivation player
- **Interactive 3D Canvas** - Drag-to-rotate, scroll-to-zoom, smooth transitions

### 🛠️ Tech Stack

- **React 18+** - UI framework
- **Three.js & React-Three-Fiber** - 3D rendering engine
- **Tailwind CSS** - Utility-first styling with dark mode
- **Lucide React** - Icon library
- **Vite** - Build tool

### 📦 Installation

```bash
npm install
```

### 🎮 Development

```bash
npm run dev
```

The app will run on `http://localhost:5173`

### 🏗️ Build

```bash
npm run build
```

### 📊 Physics Engine

- **Coulomb's Law**: F = k × (q₁ × q₂) / r²
- **Superposition Principle**: F_net = ΣF_i
- **Electric Field**: E = F / q₀
- **Force Vector Scaling**: Dynamic visualization of force magnitudes

### 🎨 UI Components

- **Left Sidebar**: Module navigation with glassmorphic design
- **Central Canvas**: 3D/2D interactive physics visualization
- **Right Sidebar**: Real-time control panel with sliders and toggles
- **Floating Cards**: NCERT revision notes and formulas
- **Derivation Player**: Step-through mathematical derivations

### 📝 NCERT Integration

Revision cards display:
- Electric Charge definition
- Electric Field Intensity
- Superposition Principle
- Coulomb's Law formula

### 🔧 Controls

- **Drag**: Rotate 3D scene
- **Scroll**: Zoom in/out
- **Sliders**: Adjust charge magnitudes and positions
- **Toggles**: Show/hide vectors, grid, auto-rotate
